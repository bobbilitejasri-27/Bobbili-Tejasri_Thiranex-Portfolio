### 📌 Project Overview
This is a personal portfolio website built for Thiranex Internship Task 1. The website showcases my profile, skills, and projects with a responsive design and interactive features.

### 🛠️ Tech Stack Used
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Django 5.0
- **Database:** SQLite3
- **Deployment:** Render.com

### ✅ Task 1 Requirements Completed
1. **HTML Structure:** Semantic HTML with proper sections - Home, About, Projects, Contact
2. **CSS Styling:** Responsive design with Flexbox/Grid, custom colors, hover effects
3. **JavaScript Interactivity:** Button click color change, form validation, smooth scroll
4. **Backend Integration:** Django models, views, templates, and Admin panel for content management

### 🚀 Features
- Fully responsive on mobile, tablet, and desktop
- Dynamic project section managed via Django Admin panel
- Contact form with backend validation
- Clean UI with modern design
